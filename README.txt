CEA Rainfall Tracking System

Developer:
Eman Tanweer Khaliq

Project Overview:
This project is a full-stack rainfall tracking system developed for the Centrala Environmental Agency. The system allows users to register, log in, manage rainfall readings, access API endpoints, and monitor API request logs.

Tech Stack:
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP
- phpMyAdmin

Main Features:
- User registration and login
- Password hashing
- API key generation
- Rainfall data CRUD operations
- Search/filter rainfall readings
- Public rainfall API endpoints
- Admin API endpoints
- API request logging
- API documentation page
- Admin logs page
- Session timeout logout
- CSRF protection
- Secure POST delete operation
- SQL injection prevention using prepared statements
- Role-based access control

Database Tables:
- users
- rainfall_data
- api_logs
